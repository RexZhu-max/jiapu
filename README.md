
  # Design Heritage Homepage Interface

  This is a code bundle for Design Heritage Homepage Interface. The original project is available at https://www.figma.com/design/UmRzdCNka0uEBKXho26eoC/Design-Heritage-Homepage-Interface.

## Running the code

1. Install dependencies:

```bash
npm i
```

2. Start backend API server (port `3001`):

```bash
npm run dev:server
```

3. Start frontend dev server (port `3000`):

```bash
npm run dev
```
